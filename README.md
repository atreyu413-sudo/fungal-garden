# Fungal Garden

A Foundry VTT module for D&D 5e that simulates a living fungal ecosystem growing on a Tortle's shell.

Originally inspired by the "Fungal Garden Thesis" created for the character Ogeez, this module transforms the paper rules into a fully automated simulation.

## Features

- Living fungal ecosystem
- Hex-grid shell simulation
- Dynamic growth and decay
- Crossbreeding and hybrid generation
- Persistent species registry
- Automated harvesting
- Foundry Actor integration
- Discovery journal
- Configurable simulation settings
